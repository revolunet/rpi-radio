# Radio rpi

Recycle an old radio using RaspberryPi and NodeJS

## Pinout

 - P1-9: LED GND
 - P1-11: LED RED
 - P1-13: LED GREEN
 - P1-15: LED BLUE

## system sound

amixer sset PCM,0 90%

amixer cset numid=3 1
